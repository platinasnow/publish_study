[4장] 웹 페이지 레이아웃

  * 웹 페이지 개요
	논문과 관련된 용도로 시작한 HTML 문서는 1995년도에 민간에 공개되며 발전

  1. 레이아웃 구분
     : 레이아웃을 만들 때 가장 중요한 것이 공간 분할 방법
       다음과 같은 순서를 지키면 된다.
	(1) 웹 페이지를 구상
	(2) 웹 페이지의 구성 영역을 분리
	(3) 구성 영역을 행 단위로 분리
	(4) 나누어진 행의 내부 요소를 분리

  2. 초기화 
      : 모든 HTML페이지의 첫번째 스타일 시트는 초기화 코드로 시작.
      : 초기화 코드는 모든 웹 브라우저에서 동일한 출력 결과를 만드는 데 사용.
      : 초기화 코드는 모두 비슷하므로 아무거나 사용해도 문제 없음.

  3. 헤더 구조 작성
      gnb : global navigation bar
       lnb : local navigation bar

  4. 웹 폰트
      :사용자가 웹 페이지에 접속하는 순간 폰트를 자동으로 내려받고 해당 웹 페이지에서 사용할 수 있게 만들어주는 기능
     (*IE8 이하에서는 로컬에서는 보안문제로 폰트적용 X, 서버에서 실행해야 폰트가 적용)

  5. 수평 메뉴

  6. 콘텐트 구성

  7. 본문 구성

  8. 사이드 탭바 구성
      : 탭바는 자바스크립트로 생성하는 것이 일반적, CSS3 선택자를 이용하여 구성해보자.

  9. 목록 구성
      : 탭 목록의 요소를 목록으로 구성해보자.
	* text-overflow : ellipsis   -> text가 너무 길 경우 ...으로 생략

 10. 푸터 구성

 11. 정리
      - 레이아웃을 행 단위로 생각하는 능력
      - 초기화 코드
      - width 속성과 margin 속성을 사용한 중앙 정렬
      - overflow 속성과 float 속성을 사용해서 레이아웃 나누는 방법

* HTML5 Shiv 플러그인을 사용하면 구 버전의 인터넷 익스플로러에서도 HTML5 태그를 인식할 수 있습니다. 핮만 본문에서 탭바를 만들면서 사용한
  복잡한 CSS3 선택자는 구 번전의 인터넷 익스플로러에서 사용할 수 없습니다. 이러한 부분을 구 버전의 인터넷 익스플로러에서 만들려면 자바스크립트를
  사용해야 합니다.
