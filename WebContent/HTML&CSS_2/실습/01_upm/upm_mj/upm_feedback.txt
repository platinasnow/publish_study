   만약 <p>안에 다른 내용의 문단으로 구문하기 위해 br을 쓴거라면
   <p>를 여러개를 나눠 사용하는게 맞음

2.<p>와 <span>구분
<p> 태그 (paragraph)  : '문단'이라는 뜻으로 문단지정 태그 기능
			p와 p사이에는 하나의 공백라인 생김
<br> 태그 (Line Break) : 단순 줄을 바꾸는 기능
<div> 태그 (division) : 영역을 나눈다는 의미, 한 페이지 내에서 영역을 지정
<span> 태그 : 폭 범위라는의미로 범위 지정 기능, 영역을 나누거나 줄 바꿈을 하지 않고 일정 범위 지정해 속성을 줌

5. css블럭요소와 인라인요소
블럭 요소는 인라인 요소 뿐 아니라 다른 블럭 요소를 포함할 수 있습니다. 반면 p 요소와 같은 블럭 요소는 다른 블럭 요소를 포함할 수 없습니다.
블럭 요소 혹은 다른 인라인 요소 안에만 쓰일 수 있으며
